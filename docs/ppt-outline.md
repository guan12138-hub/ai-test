# 答辩AI相关问题准备

## 问题1：AI大模型基础概念
**问题**：请解释什么是大语言模型（LLM）？在本项目中是如何应用的？
**回答要点**：
- 大语言模型是基于Transformer架构、通过海量文本训练的深度学习模型
- 能够理解和生成自然语言
- 本项目通过Coze平台调用LLM实现：智能问答、文章生成、内容分析
- 使用提示词工程（Prompt Engineering）引导模型输出特定格式内容

## 问题2：Coze平台工作流
**问题**：请介绍你们是如何使用Coze平台构建AI机器人的？
**回答要点**：
- 在Coze平台创建Bot，设置人设和知识库
- 配置Bot的技能：对话、知识检索、插件
- 通过Coze API（/v1/chat）与Bot交互
- 设置了食材管理相关的提示词和回复模板

## 问题3：AI接口调用逻辑
**问题**：请说明系统中AI功能的后端调用流程。
**回答要点**：
1. 前端发送请求到后端Controller
2. Controller调用CozeAiService
3. Service构建HTTP请求，携带API Key
4. 调用Coze API的/chat或/images/generations
5. 解析返回的JSON数据
6. 将结果存入数据库（ai_conversations/ai_generated_contents）
7. 返回给前端展示

## 问题4：数据安全与隐私
**问题**：AI功能如何处理用户数据的安全性？
**回答要点**：
- API Key存储在服务端配置，不暴露给前端
- 用户对话记录保存在本地数据库
- 可设置数据保留期限
- 遵循最小数据原则，只传输必要信息到AI服务

## 问题5：AI能力评估
**问题**：如何评估AI模块的效果？
**回答要点**：
- 问答助手：测试标准问题集，评估回答准确性
- 文章生成：人工评分内容质量和相关性
- 绘图生成：视觉满意度评价
- 分析报告：与实际数据对比验证
